# vscode-base64
Quickly base64 encode/decode your text selections.

![Screenshot](https://www.dropbox.com/s/3ep3o6a3vkb5gco/vscode-base64.gif?raw=1)

## Commands:

* **Base64 Encode** 
* **Base64 Decode**

## Keybindings:

* Mac
    * **cmd+e cmd+e** Encode
    * **cmd+e cmd+d** Decode
* Win/Linux
    * **ctrl+e ctrl+e** Encode
    * **ctrl+e ctrl+d** Decode
